# 使用工具和技能

本教程介绍如何在 DeerFlow 中配置和使用工具（Tools）与技能（Skills），让 Agent 能够访问文件操作和特定领域能力。

## 配置工具

在 `config.yaml` 中添加工具：

```yaml
tools:
  # 沙箱文件操作
  - use: deerflow.sandbox.tools:ls_tool
  - use: deerflow.sandbox.tools:read_file_tool
  - use: deerflow.sandbox.tools:write_file_tool
  - use: deerflow.sandbox.tools:bash_tool
```

## 启用技能

通过 DeerFlow 应用界面的技能面板启用技能，或直接编辑 `extensions_config.json`。

示例：启用代码文档技能

1. 打开 DeerFlow 应用
2. 点击侧边栏中的扩展/技能图标
3. 找到 `code-documentation`，切换为启用

## 使用技能

启用技能后，在对话中选择该技能，然后发送请求：

```
为这个项目的认证模块生成文档
```

Agent 将按照技能的工作流完成任务。
